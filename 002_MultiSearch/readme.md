# How to use
## 1.Prepare
* PDF file in same directory
* write keywords you wanna search in "dict.txt"
## 2. cmd
 > python pdf2txt.py (pdf file name).pdf
 
 For example,
 > python pdf2txt.py text.pdf

If keywords in pdf file, the lines written in "hitword.txt".
